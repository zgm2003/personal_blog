# 关于本站

这是 **智澜管理系统** 的项目文档站，记录系统的架构设计、模块实现和更新日志。

## 文档分类

按模块组织，每个模块包含前后端完整实现：

| 分类 | 内容 |
|------|------|
| **概览** | 项目简介、技术栈、快速开始 |
| **架构设计** | CMVD 分层、BaseModule、useTable Hook、N+1 优化 |
| **AI对话** | 智能体管理、SSE 流式输出、Redis 异步队列、模块概览 |
| **用户认证** | 多端登录、Token 双令牌、权限管理、RBAC 模型 |
| **系统管理** | 文件上传、多云存储、系统设置 |

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端框架 | Webman (PHP) |
| 前端框架 | Vue3 + TypeScript |
| UI 组件 | Element Plus |
| 数据库 | MySQL + Redis |
| 构建工具 | Vite |
| 文档站 | Astro |
